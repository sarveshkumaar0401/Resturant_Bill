# Resturant_Bill
A simple React-based restaurant application with the following features:

User Login system.
Menu browsing with categories (Main Course, Appetizers, Beverages).
Add/remove item quantities.
Generate a bill dynamically.
UPI QR code payment integration (works with GPay, PhonePe, Paytm, etc.).
Thank You screen after successful payment.
